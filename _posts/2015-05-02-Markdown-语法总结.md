~~~
1. [word](url) 超链接
2. **First part** 加粗
3. {% highlight YAML %} …… {% endhighlight %}  YAML代码变色
4. -
    -      列表
5. {% highlight HTML %} …… {% endhighlight %}  HTML代码变色
6. <sup>[[1]](#[1])</sup> 超链上角标
7. <ul style=‘list-style-type:none;’>
    <li id=“[1]”> [1], </li>                超链上标的
    </ul>
8. <strong>Part3</strong> 加粗
9. <q>aboutme.html</q> 方括号（红框内白）
10. <code>_include</code> 代码强调
12. {% highlight Bash shell scripts %} … {% endhighlight %} bash代码变色
13. * Table of Contents
      {:toc}        加入文章目录
14. ~~~…..~~~ 强调
15. ![Elaphurus davidianus](https://i.imgur.com/Mdc4szJl.jpg “Père David’s deer”) 图片
16. *This is emphasized* 斜体强调
17. H<sub>2</sub>O 下角标
18. <cite>(That’s a citation)</cite> 斜体
19. <u>Underline</u> 下划线
20. <del>HTML and CSS are our tools</del> 横划线
21. <center> — Idylls of the King, Tennyson </center> 居中
22. > Justification:
      > This species is…. 凸显
23. ### Ordered Lists   带序号的列表和不带序号的列表
     1. Item one
          1. sub item one
          2. sub item two
          3. sub item three
     2. Item two
     ### Unordered Lists
     * Item one
     * Item two
     * Item three
24. 表格
      | Kingdom | Phylum | Class | Order | Family |
      |:———:|:———:|:———:|:———:|:———:|
      |ANIMALIA|CHORDATA|MAMMALIA|CETARTIODACTYLA|CERVIDAE|
25. {% highlight css %}… {% end light %} css 代码高亮
26. {% highlight ruby linenos %}… {% end light %} ruby 代码高亮
27. $$\LaTeX$$ 加入latex
 28.  \<q\> tag 不知道是什么
29. <sup>[1]</sup> 上角标
30. [1] **Pitra, C.; Fickel, J.; Meijaard, E.; Groves, C. (1 December 2004).** [“Evolution and phylogeny of old world deer”](http://arts.anu.edu.au/grovco/pitra%20deer.pdf). *Molecular Phylogenetics and Evolution* **33** (3): 880–895. 参考文献注释
~~~
